# Global Trends Analysis using Gapminder Dataset

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Gapminder dataset to study global trends in:

- Population
- GDP per capita
- Life expectancy

The objective is to uncover relationships between economic development and health outcomes across countries and continents.

---

## Dataset
Dataset used: Gapminder World

Columns:
- country
- continent
- year
- lifeExp
- gdpPercap
- pop

---

## Project Tasks
- Data loading and preprocessing
- Pivot table analysis
- GDP percentile analysis
- Life expectancy categorization
- Top GDP countries visualization
- Regex country search
- Boxplots
- Correlation heatmap
- Trend analysis
- Country comparison (Afghanistan vs Switzerland)

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Results
Key findings:
- Strong positive correlation between GDP and life expectancy
- Global life expectancy improved steadily from 1952 to 2007
- Developed countries show better health and economic indicators
